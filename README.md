# VeriForge

Research prototype exploring a natural language to DSL pipeline for 
zero-decoherence LLM roleplay. A meta-questionnaire elicits worldbuilding 
specifications in natural language; these are translated into a 
machine-readable VeriForgeDSL artifact that enforces world rules 
deterministically at LLM inference time.

Current status: hypothesis-stage research. All architectural claims are 
unverified propositions under active investigation.
